## Factors

A function to get factors of a given input

### Usage

```
npm install --save factors
```

```js
let factors = require('factors');

factors(16); // [ 1, 2, 4, 8, 16 ]
factors(178990); /* [ 1, 2, 5, 7, 10, 14, 35, 70, 2557, 5114, 12785, 17899, 25570,
                      35798, 89495, 178990 ] *
```
